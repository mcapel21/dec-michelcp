  1. 1. Què és Vanilla Javascript?

    • JavaScript Vanilla es sense cap tipus de framework o llibrería es a dir JavaScript pur.

  2. 2. Diferència els ECMA Script que pertanyen a Javascript i a Javascript Modern

    • Els ECMA Script antics son del ES1-ES5, pertanyen al JavaScript modern ES6-ES12

  3. 3. Qui va crear Javascript? De quina empresa era?

    • Creat per Brendan Eich, pertanyia a l'empresa Netscape

  4. 4. Quina empresa hi ha actualment darrera de Javascript?

  5. 5. Què és un arrow function? A quin ES apareix (quin EcmaScript)

    • Es una funció on es reduida i es com un this no vinculable. Va apareixer a E6.

    • Ex: const sum = (a, b) => (a + b);

  6. 6. Què és un callback? posa un exemple.

    • Una funció callback només és definxeis, es crida i s'executa sempre i quan únicament hi passa un event en concret.

    • Per exemple l'execució d'un botó que fa una funcío (escriure per consola)

  7. 7. Què és una promesa (promise) en Javascript? Posa un exemple. Explica també la diferència entre la programació síncrona i la programació asíncrona, posa un exemple.

    • Una promesa (promise), és utilitzat per computacions asincronas, representa un valor que pot existir o no.

    • Programació síncrona: Les declaracions es completen abans de que s'executi la següent. En aquest cas, el programa s'evalua exactament en el ordre de les declaracions i l'execució del programa si es para o si una de les declaracions tarda molt.

    • Programació asincrona: Les funcions asincrones normalment accepten una devolució de crida com a parametre i l'execució continua en la següent linea immediatament després d'invocar a la funció asincrona. La devolució de la crida només s'invoca quan l'operació asincrona es completa y la pila de crides es buida. Operacions com carregar dades d'un servidor web o BBDD, es tenen que realitzar de forma asincrona ja que el fil principal pugui continuar executant altres operacions en comptes de bloquejar fins que es termini una llarga operació ( a nivell navegador es bloquejaria la UI).
